# Personal vim configuration folder

It is my personal vim configuration folder.
Help and advices of any type are welcome.

## Installation

```bash

git clone https://github.com/Wykiki/vim-config.git ~/vim-wyki && \
cd ~/vim-wyki && \
./install.sh

```
